    对遍历Map的几种方法进行性能测试

    最近在工作中注意到Map遍历的性能差异，周末自己进行了测试，对Map的便利方式进行归纳总结。

    准备工作

    因为String作为Value不具有代表性，定义Student对象，测试数据量较大情况

    第一种方法 使用iterator遍历EntrySet

    第二种方法 使用forEach循环遍历EntrySet

    第三种方法 使用JDK1.8中的Lambda表达式形式的forEach遍历EntrySet

    第四种方法 使用for循环KeySet取值

    测试结果

    从测试结果来看，第二种方法遍历Map的效率最高，但是第二种方法和第三种方法只是使用了不同形式的for循环，耗时却相差悬殊，关于Lambda表达式为何会造成遍历集合效率降低的问题，在以后的文章中再进行详细分析讨论。
